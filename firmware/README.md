# AZ-M5ortho ファームウェア
AZ-M5ortho で使用しているファームウェアソースです。<br>
こちらのソースは個人で利用する場合には自由にご利用いただいて大丈夫です。<br>
二次配布については非営利であれば著作権表示をして頂ければご利用して頂いて大丈夫です。<br>


<br><br>

<br><br>

## 対応商品

- AZ-M5ortho


## 開発環境

- Arduino 1.8.13
- M5Core2 0.0.2
- ESP OTA 0.0.3
- NimBLE-Arduino 1.0.2
- ArduinoJson 6.18.4
- lv_arduino 3.0.1
- Adafruit NeoPixel 1.7.0
